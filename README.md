# Final502

# National Resilience: The Intersection of Trust, Poverty, and Demographics"

## Project Theme
The theme of the Tripartite Model is the exploration of global systemic vulnerabilities at the intersection of macroeconomic stability (Wealth), institutional integrity (Trust), and aging populations (Demographics). 

## Data Sources (name)
The data for the code is contained in these three files:
1. `API_NY.GNP.PCAP.CD_DS2_en_csv_v2_230.csv` (Wealth / GNI Per Capita)
2. `ti-corruption-perception-index.csv` (Trust / Corruption Perceptions Index)
3. `POP_DPND.csv` (Demographics / Old Age Dependency Ratio)

## Visualizations Overview
For this project, I worked on a total of 5 charts: 3 built with Python and 2 built with R.

### Python Charts (Interactive)
The Python code was written and uploaded from Google Colab. These three visualizations were designed to be fully interactive:
* **Facet Line Chart**
* **Bubble Chart**
* **Boxplot**

**Outputs:** The static outputs for these charts are available as 3 `.pdf` files, alongside 3 video clips. 
*Note to the audience:* Please view the provided clips to see exactly how interactive the charts are!

**Interactive Exploration:** Feel free to run the Python code yourself if you want to investigate the interactive functions more closely. Just remember to change the data file paths in the script to match where you have saved the CSV files on your own drive or local machine.

### R Charts (Static)
The R visualizations are static and do not contain interactive elements:
* **Ridgeline Plot**
* **Barplot**

**Outputs:** The outputs for these charts can be found as `.png` image files.
